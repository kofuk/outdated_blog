About KoFuk

GNU/Linux 大好きでアンチ GUI (これは誇張) の情報系の学生です。
お気に入りのエディタは Emacs で、高校生のころから使っています(蛇足)。 Emacs じゃないと能率が上がらない。というかこれは自分の設定ファイルが導入済みの Emacs じゃないと能率が上がらないということだと思う。まあ別に Vim でもいいんだけど。というか Emacs の前普通に Vim 使ってたし。でも方向キーがどうも苦手で、方向キー問題を解決するために Emacs を使ってるまである。だから最近の Visual Studio Code みたいなやつはどうも苦手。
高校時代は友人と共同でウェブサービスを作り、その一環でよく Android アプリを作っていました。
今[いつ？]は主に UNIX システムプログラミングに興味があります。
最近ブログを自分のサイトで管理するのが面倒なってしまった (老いを感じずにはいられない) ので、 はてなブログに移行しました。というか移行中。
あともう一つ、老いを感じることは、この Web サイトを作るのさえ面倒だなと思ってしまうことですね。 じゃあなんでやってるんだ、という話ではあるけれど。
